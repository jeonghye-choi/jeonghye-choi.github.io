## Deploy

배포 전 확인 사항

1. Github 저장소 이름이 username.githube.io가 맞는지 확인

2. ./_config.js 에서 siteUrl을 올바르게 작성했는지 확인

3. 브랜치 확인! (master이외의 브랜치)

```
$ npm run deploy
```
